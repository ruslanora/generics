# Security Policy

**PLEASE DON'T DISCLOSE SECURITY-RELATED ISSUES PUBLICLY**

## Supported Versions

Only the latest major release of the library receives bug fixes and updates.

## Reporting a Vulnerability

If you discover a security vulnerability, please report it privately using [GitHub's security advisories](https://github.com/ruslanora/generics/security/advisories). All security vulnerabilities will be promptly addressed.
